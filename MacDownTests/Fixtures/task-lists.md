# Task List Tests

## Basic Task Lists

- [ ] Unchecked task
- [x] Checked task
- [ ] Another unchecked task
- [X] Checked with capital X

## Nested Task Lists

- [ ] Parent task
  - [x] Completed subtask
  - [ ] Incomplete subtask
  - [x] Another completed subtask
- [x] Completed parent

## Mixed with Regular Lists

- Regular list item
- [ ] Task item
- Another regular item
- [x] Completed task
- Final regular item

## Tasks with Inline Formatting

- [ ] Task with **bold text**
- [x] Task with *italic text*
- [ ] Task with `code`
- [x] Task with [link](https://example.com)

## Numbered Task Lists

1. [ ] First task
2. [x] Second task (completed)
3. [ ] Third task

## Edge Cases

- [] Invalid (no space)
- [ ] Valid unchecked
-[ ] Invalid (no space after dash)
- [x ] Invalid (space before X)
- [ x] Invalid (space after bracket)
