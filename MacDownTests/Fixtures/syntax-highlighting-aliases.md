# Language Aliases Test

## JavaScript (js alias)

```js
const greeting = "Hello, World!";
console.log(greeting);
```

## Objective-C (objc alias)

```objc
NSString *greeting = @"Hello, World!";
NSLog(@"%@", greeting);
```

## HTML (maps to markup)

```html
<!DOCTYPE html>
<html>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
```

## Shell (sh alias for bash)

```sh
#!/bin/sh
echo "Hello, World!"
```

## C++ (c++ and cpp aliases)

```c++
#include <iostream>
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```
