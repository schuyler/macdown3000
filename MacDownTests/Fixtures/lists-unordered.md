# Unordered List Tests

## Basic List with Asterisks

* First item
* Second item
* Third item

## Basic List with Hyphens

- First item
- Second item
- Third item

## Basic List with Plus

+ First item
+ Second item
+ Third item

## List with Paragraphs

- First item with text

- Second item after blank line

- Third item

## List with Inline Formatting

- **Bold** item
- *Italic* item
- `Code` item
- [Link](https://example.com) item

## Tight vs Loose Lists

Tight list:
- Item one
- Item two
- Item three

Loose list:

- Item one

- Item two

- Item three
