Language Aliases Test

JavaScript (js alias)

const greeting = "Hello, World!";
console.log(greeting);

Objective-C (objc alias)

NSString *greeting = @"Hello, World!";
NSLog(@"%@", greeting);

HTML (maps to markup)

<!DOCTYPE html>
<html>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

Shell (sh alias for bash)

#!/bin/sh
echo "Hello, World!"

C++ (c++ and cpp aliases)

#include <iostream>
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
