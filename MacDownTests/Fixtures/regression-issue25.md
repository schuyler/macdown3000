# Adjacent Shortcut Links (Issue #25)

Two shortcut-style links side by side should both render:

[foo] [bar][baz]

A shortcut link before a reference link:

[first] [second][ref]

Multiple shortcut links in sequence:

[one] [two] [three]

Shortcut link at end of paragraph before another on next line:

[alpha]
[beta]
